CSS3 introduces advanced styling features such as gradients, shadows, transforms, and transitions.
Gradients: Create smooth color transitions.
Shadows: Add depth and dimension to elements.
Transforms: Rotate, scale, skew, and translate elements in 2D or 3D space.
Transitions: Apply smooth animations to CSS properties.
Example: Adding gradients and shadows to elements, transforming elements on hover.
